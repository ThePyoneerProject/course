# The Pyoneer Course

The Pyoneer Course from [The Pyoneer Project](https://pyoneers.dev/) is an open-source curriculum for learning full-stack web development.

Lessons are structured through a combination of original written content and a compilation of carefully curated resources from the web.

This repo contains the actual lesson files used on the website platform.

The community can be found on our [Discord server](https://discord.gg/TSMacaF9SG).
